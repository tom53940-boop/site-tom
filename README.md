# Finanssia static website

## Modifier le contenu

Le contenu est maintenant directement dans les pages HTML.

- Accueil: `index.html`
- Pages principales: `services/`, `demarche/`, `prendre-rendez-vous/`, `recrutement/`, `contact/`, `mentions-legales/`

## Configurer le formulaire de contact

Dans `contact/index.html`, remplacez `YOUR_WEB3FORMS_ACCESS_KEY` par votre cle Web3Forms.

## Arborescence

- `index.html`: page accueil
- `services/`, `demarche/`, `prendre-rendez-vous/`, `recrutement/`, `contact/`, `mentions-legales/`
- `assets/css`: styles
- `assets/js`: scripts
- `sitemap.xml`, `robots.txt`
